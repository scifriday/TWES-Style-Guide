Logo Treatments

The Will Edwards Show identity is versitile enough to use in a variety of media type, form video to digital to print. However, each has specific usages that must be followed to maintain the logo's integrity.

Main Logos

Stacked

The main logo is the stacked version with the martini brand mark. This should be used the most.

Horizontal

There is also a horizontal version of the main logo for use as a webstie masthead or in narrow banner ads.

On Black

Secondary Logos

In applications already featuring the brand mark, the secondary usage would be appropriate. These should not be used on black as is; use the knockout style instead.

Type Only

This stipped down version is available when the brand mark appears nearby already, but the boomerang/trapezoid background cannot work within a design.

Brand mark

The brand mark is a representation of one of Will's most popular characters, and her signature drink. It can be used alone, but only if a secondary logo style is used nearby.

Initials

The TWES initial treatment corresponds to community related promotions, like "WE Funny", where Will's initials are used as a title. This is to be used very rarely, and only in the communtiy context.

Black & White / Grayscale Logos

These can be used in single color applications, such as newsprint.

Knockout

The knockout style works best on bright, solid background colors.

Minimum White Space

The border line shows the minimum amout of white space that should surround the logo. It is approximately the width of the "W".
